平庸也是人生

发表于 2016-11-28

黄昏街头漫步，视线沿着街道由近到远处，最后汇成一点。楼盘从近到远呈现出阶梯排列。物体的渐变很容易看得到。

然而散步去的时候还有落日余晖，到回来的时候已经是华灯初上，期间不到半个小时，不知不觉地变暗，时间的变化那么的自然。 想起丰子恺的散文《渐》：“空间的渐变很容易看出。由夏一天天、一时时、一分分、一秒秒地移向冬，其间实在没有显著的痕迹可寻。”

只是平庸的人毕竟是占多数的，如陈奕迅在《浮夸》里的一句：“在世间平凡又普通的路太多，屋村你住哪一座”，所以平庸并不可怕。
平庸对于人生只是个结果，而人生最大的意义是过程，我们要在乎的是这个过程，我们要在渐变的过程中感受到人生之美，而不是出于某种所谓的远大理想，牺牲大部分人生过程的乐趣去取得一个结果。也许很多人愿意吃上几天泡面，而为了吃一顿奢侈的山珍海味。而我个人喜欢细水长流的时光里，每天就着青菜豆腐，番茄炒蛋轮着来。

从王思聪驳斥冯导所感

很早在微博上就关注了王思聪，他颠覆了我对于富二代都是纨绔子弟的印象。
从他平时对公共事件的评论，包括这次直接反呛冯小刚，并且驳斥很到位，他不仅仅是有钱的富二代，并且受到过良好教育，在思想和价值观的精神层面上也不缺乏。
想想当我们挣扎在底层的，接触更多的是人性的阴暗面,花绝大多时间在想的是如何生存？如何买房？如何让小孩上好学校。王思聪这位哲学生有大把时间思考什么是生活。
他不仅是有钱，还有思想，有话语权，他的家庭条件让他很容易获得先进教育。并且将来他的小孩必然有优越的教育环境，形成良性循环，社会阶级趋于固化也就成为可能。如果现在有人问我：“王侯将相宁有种乎？” 我的回答是肯定的。
再回到这次王思聪微博里面驳斥冯导，王说这是“本来两家私企的恩怨应该是两家公司的事情”、认为冯导不应该用道德层面上煽动网络舆论。并且王还说了“没有人和钱过不去”。王的言下之意就是告诉冯导在商言商，大家应该遵守自由市场经济的规律，按契约精神办事就行了。王驳斥的可谓有理有据，当然万达真是完全把全国影院都垄断了，那又另当别论。

树莓派-raspberry-pi-3

发表于 2016-11-14   |   分类于 Linux

家里电脑主要装win系统，写软件要用linux，用vmware装了虚拟机，总感觉没有这么方便，每天经常要开关机，有时躺床上想手机登录下。偶然逛知乎发现了树莓派(raspberry pi)这么个玩意，于是到某宝淘了树莓派(raspberry pi)3回来，感觉太实用了。

这小东西长宽跟银行卡片差不多。但麻雀虽小，五脏俱全。

直接用个手机充电器或者移动电源就可以供电，蓝牙、LAN口、wifi俱全，并且跑linux（Debian）系统，直接用张SD卡安装系统，可以再扩展一个移动硬盘存放海量高清片（你懂的）。
关键功耗很小，完全跟路由器一样可以全天侯开机。

有网络有系统，能玩的花样就多了,可以做个私有云（owncloud）、NAS服务器、笔记同步、git服务器、离线下载，自己个人用时绰绰有余了，自己平时喜欢看书，收藏好多电子书，准备用PHP在树莓派上做个在线阅读系统，随时掏出手机和kindle可以连接上去在线看。带上移动电源往背包一塞，就是个移动服务器了。

照一架荼蘼如雪

发表于 2016-11-14   |   分类于 随笔

“点火樱桃，照一架荼蘼如雪。”

这是辛弃疾的《满江红》里面的第一句，纯粹写景：火红的樱桃，照在白色荼蘼花上更加雪白。

樱桃和荼蘼不因一方的照耀而使另一方失色，相反彼此互相促进，更加醒目，整体更加唯美。它们是平等的，是积极的，互相促进的。

莫言曾说过：“文学和科学相比，的确没什么用处，但文学最大的用处，也许就是它没有用处。教育也如此，所谓的分数、学历、甚至知识都不是教育的本质，教育的本质是：一棵树摇动另一棵树，一朵云推动另一朵云，一个灵魂唤醒另一个灵魂。”

其实人与人之间也似樱桃和荼蘼之间，润物细无声般互相照映，互相促进。即时我解读此词并非辛弃疾原文本意，但王国维的人生三境界分别引用三篇宋词里的句子，也非原文的意思，而是他借这些词赋予了新意义。

好的诗词甚至好的文学都很少直接说教讲理，但是我们却能在澹泊、隽永的词句中感受到一种精神和力量。
