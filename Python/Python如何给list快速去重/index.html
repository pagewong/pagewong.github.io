去重的方法

去重的方法其实有很多种，但是最优最快的是哪种呢？
我把想到和搜索到的去重的方法,
分成“保持原有顺序”和“不保持原有顺序”两类，列举如下：

- 保持原有顺序

def f2(seq): 
    checked = []
    for e in seq:
       if e not in checked:
           checked.append(e)
    return checked

def f4(seq): 
    noDupes = []
    [noDupes.append(i) for i in seq if not noDupes.count(i)]
    return noDupes

def f6(seq, idfun=None): 
    if idfun is None:
       def idfun(x): return x
    seen = {}
    result = []
    for item in seq:
        marker = idfun(item)
        if marker in seen: continue
        # python2用以下代替:
        # if seen.has_key(marker): cntinue
        seen[marker] = 1
        result.append(item)
    return result

- 不保持原有顺序

def f1(seq):
    set = {}
    map(set.__setitem__, seq, [])
    return set.keys()

def f3(seq):
    keys = {}
    for e in seq:
       keys[e] = 1
    return keys.keys()

def f5(seq):
    set = Set(seq)
    return list(set)

最优方案

首先测试运行速度比较:

f1: 0.18
f3: 0.17
f5: 0.19

f2: 13.24
f4: 11.73
f6: 0.37

综合速度和性能，在能“保持list原有顺序”下，很显然，方法F6是最快的。
